Layer-2-Network-Plan-Validator
==============================

What is it?
-------------

The Layer 2 Network Plan Validator is a network management tool. It has been designed to aid in the management of the switch topology at the University of Aberystwyth. As a project, the developer aims to maintain a robust standards-based tool with freely available source code.
